# Rag-chatbot_Sid
This a rag chatbot which answers questions based on the documents you provide(Webscraping.txt and oops_java.txt is already added)

---Step By Step Guide to run this project-----

1.Create your python environment and install required libraries(available in requirements.txt)

2.to run main.py use the command:

 --> cd backEnd
 
 --> uvicorn main:app --reload
 
3.to run app.py use the command:

 --> streamlit run app.py
